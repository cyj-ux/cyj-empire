Commandos para desemsamblar archivos


comando objdump:

objdump -drwC -Matt,i8086 foo.o > foo.dump
# desemsambla en 16 bits, sólo archivos objeto y lo vuelca a un archivo .dump

objdump -drwC -m i8086 foo.o
# desemsambla en 16 bits, sólo archivos objeto y lo imprime en pantalla

objdump -D -Matt,i8086 -b binary -m i386 foo.bin
# para código de 16 bits

objdump -D -Matt,i386 -b binary -m i386 foo.bin
# para código de 32 bits

objdump -D -Matt,x86-64 -b binary -m i386 foo.bin
# para código de 64 bits

objdump -D -Matt,i8086 foo.bin
# disemsambla el archivo entero.

objdump -d -Matt,i8086 foo.bin
# disemsambla sólo las partes de código

objdump -D -b binary -mi386 -Maddr16,data16 foo.o
# desemsambla con addr y data
